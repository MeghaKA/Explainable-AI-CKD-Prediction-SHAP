# 🧠 Explainable AI for Diabetes & Chronic Kidney Disease Prediction

## 🎯 Overview
This project focuses on predicting the risk of **Diabetes and Chronic Kidney Disease (CKD)** using Machine Learning models, and enhancing transparency using **SHAP (Explainable AI)**.

The objective is not only to build accurate predictive models but also to understand *why* the model makes certain decisions — a key requirement in healthcare applications.

---

## ⚙️ Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- SHAP (Explainable AI)  

---

## 🚀 What I Built
- Developed machine learning models for:
  - Diabetes prediction  
  - CKD prediction  
- Performed data preprocessing and handling missing values  
- Applied SHAP to interpret model predictions  
- Visualized feature importance and impact

## 🧠 Key Insight
The model identifies several clinical features as important predictors of disease risk.  
Higher values in certain health indicators contribute positively to predicted risk.

---

## 💡 Why This Matters
In healthcare, predictions alone are not enough — models must be interpretable and trustworthy.

This project demonstrates how Explainable AI can support better understanding of machine learning decisions in medical contexts.
